    # 技能清单

    # 编程语言

    • 在项目中使用 HTML 和 CSS，高度还原设计稿
    • 熟练使用 JavaScript/TypeScript，熟练掌握其基础，能够灵活运用、高效开发

    # 工作经验

    # 上海魔航科技 2023 年 02 月 ~ 2023 年 03 月

    # 业务支持

    独立开发: 独自负责公司平时运营活动中需要用到的落地页，参与项目需求讨论，并进行项目的技术选型，使用 React+antui+webpack+less+rem 构建开发框架，快速开发活动落地页。

    # 上海达龙信息科技 2021 年 05 月 ~ 2022 年 12 月

    # PC 端、移动端达⻰云游戏项目重构

    独立开发: 独自负责达⻰云游戏连接使用中的前端项目重构，通过 ts + composition api 的形式重写并统一维护到新的组件库中，拆分开各个功能模块，便于高效维护代码。同时对项目的连接进行优化，通过 WebSocket 将连接和账号助手的通信进行速度优化。将连接过程(通过多次握手获取各个阶段状态 key 值完成 webrtc 的整个连接过程) 拆分出来，实现代码的高度利用率(各客户端连接方式相同)，使连接速度更快。
      负责产品的性能优化，以及性能问题排查与解决。解决项目存在连接时间过长或经常卡顿问题，更新了 webrtc、帐号助手和前台推送连接方式，对 websocket 和 webrtc 连接进行封装，减少了代码量，且便于优化，使连接加载过程更加快速，减少了连接卡顿问题，增加了平台客流量。平台云游戏加载时间从 15s 优化到 5s，出现的连接卡顿情况也极速减半。
      将 mixin 混合开发用 composition api 代替，利用函数导入可以实现模块的复用，优化了代码的可读性，更便于后期维护，更能精准定位问题，减少维护时间。
     所负责项目荣登公司年度项目榜单。

    # 业务支持

    业务开发: 应公司产品需求，需要对云游戏的一些功能进行优化和添加。添加了游戏的物理手柄操作、虚拟手柄/键盘模拟操作、手指触控模拟鼠标点击等，提供了平台云游戏更多的可玩性，为平台增加了客户量。
     并参与解决了平台的云游戏连接以及使用过程的卡顿问题。

    # 学而思-好未来集团 2019 年 05 月 ~ 2020 年 09 月

    # 学而思落地⻚的活动组件库搭建

    团队开发: 从 0 搭建学而思落地⻚需要使用的一些组件库。由于平时学而思需要推出一些活动来吸引用户提高用户量，研发一些落地⻚平时需要使用的组件库(表单库、提示框、加载等)。对于业务组件，组件库基于 json- schema 定义，贯彻组件化开发(避免组件耦合)的思想，遵循“简单接入、按需使用”的原则，只需要提供必要的 参数就可以帮助其开发。
      负责公司内部组件库、函数库的开发与维护。实现了表单登录注册、提示框、加载动画、底部导航栏等活动常用模块，使用 webpack 实现组件可按需加载，通过单元测试优 化功能使组件更加安全。
      该组件库广泛用于各个落地⻚项目中。

    # 商城及其服务管理系统

    独立开发: 为促进用户消费，团队推出礼品商城，当用户购买课程获得积分或者通过参加活动的方式获得商城积分之后，就可以在商城领取礼品。为方便运营人员运营商城，配合活动推出新礼品或者在商城添加活动时，可以通过商城服务的后台管理系统进行操作。
      使用 VantUI 和 ElementUI 配合 VUE 进行前端页面开发，实现了 H5 微信授权分享、权限管理、活动发布、商品管理、信息导入、积分管理等主要功能。利用混合开发配合组件化开发，大幅度减少了代码量。
      该系统应用于各个日常运营活动中。

    # 业务支持

    独立开发: 由于团队需要吸引更多的客户量，所以需要经常推出各种活动，或团队自设计的活动，或其他学而思发布的活动，根据不同的研发环境，调整团队使用框架(主要为 Vue 和 React)，为团队提供各种活动上的业务支持。
      构建了 vue+webpack+vantui+less+rem 移动端开发解决方案。编程高度组件化，优化登录逻辑，优化了图片请求慢，时效性不高的接口进行数据缓存，canvas 海报图缓存等。
      解决了技术栈存在的首屏加载过慢的问题，从 7s 优化到 2s，负责产品的性能优化，采用图片懒加载、加载骨架屏、虚拟滚动、CDN 存储等优化方式，提升了用户体验。
